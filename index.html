UMass Robotics Club Battlebot

This code allows an ESP32 to control a dual-motor battlebot vehicle using a Bluetooth game controller via the Bluepad32 library. It reads throttle, brake, and steering inputs from the controller and converts them into smooth motor commands using PWM signals that drive two ESC-controlled motors. The system includes ramped acceleration, steering mixing, and safety handling for controller disconnects. It also initializes and arms the ESCs, maps controller inputs to motor duty cycles, and provides an emergency neutral braking function.
